# Organizacion de entregas estudiantiles

Este repositorio conserva los materiales de clase en las carpetas `Week_*` y agrupa las entregas en `Entregas_Estudiantes/<Estudiante>/Momento_XX`.

Reglas usadas:

- Las entregas individuales se ubicaron en la carpeta del estudiante identificado.
- Las entregas grupales se copiaron en la carpeta de cada integrante para facilitar la revision individual.
- Los archivos cuyo autor no se pudo confirmar quedaron en `Entregas_Estudiantes/Sin_Identificar/Pendiente_Identificar`.
- El contenido de los archivos no fue modificado; solo se reorganizaron rutas.
